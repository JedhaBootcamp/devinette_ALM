Projet de création d'une devinette

la machine choisit aléatoirement un nombre entre 1 et 10, ensuite, l'utilisateur tente de deviner ce nombre. 

Le code contraint l'utilisateur de ne pas choisir un nombre en dehors de cet intervalle et génère bonne réponse, sinon mauvaise réponse si son choix diffère de celui de la machine.
